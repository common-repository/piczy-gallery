Integrate your Piczy gallery into your wordpress website, free and very easy!

== Description ==

Piczy.net is a gallery website where you can easely create your own private galleries. The website is een social media based platform, so you can communicate with other users, share your gallery, and reply and like pictures.
To help our users integrate the gallery into there own website, we created this wordpress plugin.

With this plugin you can very easy show a nice gallery in your website. Automatically it gives users the opportunity to reply, and love pictures.

To use this plugin you need an account on http://www.piczy.net/

After registering you can create a gallery with groups and pictures. 
You also will find the API id and key you need to let the wordpress plugin connect to Piczy.net

If you have any questions about our website or the plugin.
Please feel free to contact us at http://www.piczy.net/contact/

Features:

 - Automatic gallery with lightbox
 - Users can reply with their Piczy account
 - Users can like with their Piczy account
 - You can choose the size of the pictures
 - You can choose the shape of the pictures
 - You can show pictures from a specific group, or all the pictures in your gallery
 - Manage your galleries and pictures with http://www.piczy.net/

== Installation ==

1. Extract piczy_wordpressplugin_vx.x.zip
2. Upload the extracted folder and files to /wp-content/plugins/
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Update the settings trought the Piczy gallery menu in wordpress
5. Add the [piczy] tag where you want to show the gallery.
6. If you want to show only a specific group, use [piczy group="< groupname >"]

Or you can search in your Wordpress admin panel to 'Piczy Gallery' and install it from their.
